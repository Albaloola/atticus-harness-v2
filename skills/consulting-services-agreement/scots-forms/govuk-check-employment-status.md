      <div data-module="govspeak" class="gem-c-govspeak govuk-govspeak govuk-!-margin-bottom-0">
    
        <div class="govspeak"><p>The Check Employment Status for Tax tool gives you HMRC’s view of a worker’s employment status, based on the information you provide. It can also be used to check if changes to contractual terms or working arrangements may alter a worker’s employment status.</p>

<p>You do not have to use the tool to make employment status decisions, but it can help you decide:</p>

<ul>
  <li>the employment status of a worker or an individual you engage or represent</li>
  <li>if the <a href="https://www.gov.uk/guidance/understanding-off-payroll-working-ir35">off-payroll working (IR35) rules </a> apply to a contract</li>
  <li>if HMRC will consider you as employed or self-employed for tax and National Insurance contributions purposes</li>
</ul>

<p>HMRC will stand by all determinations given by the tool, as long as the information you give remains accurate and in accordance with our guidance (see results section). You can use the tool again if:</p>

<ul>
  <li>there has been a change to an existing contract or service agreement</li>
  <li>the information you originally provided is no longer accurate</li>
</ul>

<p>You do not need to know who the worker is to use the tool ― you’ll still get a result. You can use the tool again when more information and a worker’s identity is known. This may enable the tool to provide a determination where the result was previously ‘unable to determine’.</p>

<h2 id="who-can-use-the-tool">Who can use the tool</h2>

<p>You can use the tool if you’re a:</p>

<ul>
  <li>hirer</li>
  <li>worker</li>
  <li>agency or third party</li>
</ul>

<h3 id="hirer">Hirers</h3>

<p>If you’re a hirer, use the tool if:</p>

<ul>
  <li>you’re directly hiring, or have directly hired, a worker</li>
  <li>you’re in receipt of a worker’s services who operates through their own intermediary</li>
  <li>there has been a change to an existing contract or service agreement</li>
</ul>

<h3 id="worker">Workers</h3>

<p>If you’re a worker, use the tool if:</p>

<ul>
  <li>you want to check the accuracy of a determination you’ve been given</li>
  <li>you’re providing your services through an intermediary to a small-sized non-public sector client and need to make a determination as part of your <a href="https://www.gov.uk/guidance/ir35-what-to-do-if-it-applies">responsibilities under the off-payroll working rules</a>
</li>
</ul>

<p>You may want to use the tool if any of these apply and there has been a change to an existing contract or service agreement.</p>

<h3 id="agency">Agencies or third parties</h3>

<p>If you’re an agency or other third party, use the tool if you want to check the accuracy of a determination you’ve been given.</p>

<h2 id="results">Results</h2>

<p>The tool provides the following determinations based on the information you give:</p>

<ul>
  <li>self-employed for tax purposes for this work</li>
  <li>employed for tax purposes for this work</li>
  <li>unable to make a determination (for employed or self-employed for tax purposes)</li>
  <li>off-payroll working rules (IR35) do not apply</li>
  <li>off-payroll working rules (IR35) apply</li>
  <li>unable to make a determination (for whether the off-payroll working rules apply)</li>
</ul>

<p>You may receive an ‘unable to determine’ result. If this happens, help will be provided.</p>

<p>HMRC will stand by all determinations given by the tool, as long as the information you give remains accurate and in accordance with our guidance. You can use the tool again if:</p>

<ul>
  <li>there has been a change to an existing contract or service agreement</li>
  <li>the information you originally provided is no longer accurate</li>
</ul>

<h2 id="before-you-start">Before you start</h2>

<p>For an engagement to be classed as employment or self-employment there must be a contract in place. More information can be found in the <a href="https://www.gov.uk/hmrc-internal-manuals/employment-status-manual/esm0543">Employment Status Manual</a>.</p>

<p>You’ll need to know:</p>

<ul>
  <li>details of the contract</li>
  <li>the worker’s responsibilities</li>
  <li>who decides what work needs to be done</li>
  <li>who decides when, where and how the work is done</li>
  <li>how the worker will be paid</li>
  <li>if the engagement includes any corporate benefits or reimbursement for expenses</li>
</ul>

<p>You may want to view <a href="https://www.gov.uk/hmrc-internal-manuals/employment-status-manual/esm11000">further guidance on using the tool</a> in a separate window to help you while you use it.</p>

<p>The tool is anonymous. We record and publish additional information showing the <a href="https://www.gov.uk/government/publications/check-employment-status-for-tax-cest-2019-enhancement">breakdown of results and the logic for the <abbr title="Check Employment Status for Tax">CEST</abbr> tool</a>. We do not collect any identifying details of those using the tool.</p>

<p>If you close the <abbr title="Check Employment Status for Tax">CEST</abbr> tool without finishing the questions, you will need to start again.</p>

<p><a class="gem-c-button govuk-button govuk-button--start" role="button" data-module="govuk-button cross-domain-tracking" data-tracking-code="UA-2.216710174.941331531.1569413497-1095782318.1566902893-Y" data-tracking-name="govspeakButtonTracker" data-ga4-attributes='{"event_name":"navigation","type":"start button"}' draggable="false" href="https://www.tax.service.gov.uk/guidance/check-employment-status-for-tax"><span> Start now </span><svg class="govuk-button__start-icon govuk-!-display-none-print" xmlns="http://www.w3.org/2000/svg" width="17.5" height="19" viewbox="0 0 33 40" focusable="false" aria-hidden="true"><path fill="currentColor" d="M0 0h13l20 20-20 20H0l20-20z"></path></svg></a></p>

<p>If you are unable to check online, you can <a href="https://www.gov.uk/government/organisations/hm-revenue-customs/contact/status-customer-service-team">contact HMRC</a>.</p>

<p>After you’ve used the tool, you’ll be able to:</p>

<ul>
  <li>save and print your answers and result for your own records</li>
  <li>use the result as a valid status determination statement</li>
</ul>
