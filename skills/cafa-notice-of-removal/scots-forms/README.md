# CAFA Notice of Removal, Scots/UK Forms

This is a US-only procedural mechanism. No Scottish or UK forms exist for removal under CAFA.

## Relevant Scottish Forms (for remit / representative actions)
- **Form G50** - Application for remit from Sheriff Court to Court of Session (Ordinary Cause Rules)
- **Form G51** - Application for remit from Court of Session to Sheriff Court
- **RCS Form 51.1** - Representative party applications (Rules of the Court of Session)
- **Consumer Scotland Act 2020** - opt-in representative action forms (awaiting commencement)
