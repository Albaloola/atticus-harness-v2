# Scottish Conveyancing, Completion Statements & Settlement Guidance

## Adapting ALTA Settlement Statements for Scotland

### Key Differences: No ALTA/HUD in Scotland

The **ALTA Settlement Statement** and **HUD-1** / **Closing Disclosure** are US-specific documents. In Scotland, conveyancing uses:

1. **Completion Statement** - prepared by the solicitor for both buyer and seller
2. **Missives** - exchange of letters forming the binding contract (not a single Purchase Agreement)
3. **Disposition** - deed transferring title (equivalent to deed)
4. **LBTT Return** - Land and Buildings Transaction Tax return to Revenue Scotland

### Scottish Conveyancing Process

| US Stage | Scottish Equivalent |
|---|---|
| Purchase & Sale Agreement | Missives (qualified acceptance → concluded missives) |
| Title search | Title search (Land Register / Sasine Register) |
| Title insurance | Searches / Land Register; no title insurance |
| Deposit / Earnest money | Typically 10% deposit (not always required) |
| Closing / Settlement | Settlement (date of completion) |
| Deed of Trust / Mortgage | Standard Security (registered with Land Register) |
| HUD-1 / Closing Disclosure | **Completion Statement** (solicitor-prepared) |
| Recording deed | Registration in Land Register |
| Transfer taxes | LBTT (Land and Buildings Transaction Tax) - payable to Revenue Scotland |
| Escrow / Settlement Agent | Solicitor acts as settlement agent (no independent escrow) |

### Completion Statement Structure (Scotland)

**Header:**
- Date of settlement / completion, Address of property, Seller(s) and Buyer(s) full names, Solicitor details (acting for each party)

**Buyer's Completion Statement:**

| Debits (Amounts Due from Buyer) | Notes |
|---|---|
| Purchase price | As per concluded missives |
| LBTT payable | To Revenue Scotland (form LBTT return) |
| Registration dues | Land Register registration fee |
| Legal fee (Buyer's solicitor) | Conveyancing fee + disbursements |
| Search fees | Land Register / property / personal searches |
| Revenue Scotland fee | £1.90 LBTT filing fee (where applicable) |
| Mortgage / Standard Security fee | Lender's legal fees if applicable |
| Survey / Home Report | If buyer commissioned additional survey |
| Anti-money laundering checks | ID verification costs |

| Credits (Amounts Due to Buyer) | Notes |
|---|---|
| Deposit already paid | Typically 10% held by seller's solicitor |
| Mortgage advance | Loan proceeds from lender |
| Pro-rated council tax | Prepaid by seller (if applicable) |
| Pro-rated factoring / service charges | If tenement |

**Net due from Buyer** = Total debits − Total credits

**Seller's Completion Statement:**

| Credits (Amounts Due to Seller) | Notes |
|---|---|
| Purchase price | As per concluded missives |
| Prepaid council tax (pro-rated) | If paid in advance |
| Prepaid factoring fees | If paid in advance |

| Debits (Amounts from Seller) | Notes |
|---|---|
| Standard Security / mortgage redemption | To lender (with redemption statement) |
| Legal fee (Seller's solicitor) | Conveyancing fee + disbursements |
| Seller's LBTT | Not applicable, LBTT is buyer's responsibility |
| Estate agent / solicitor-only fees | If using a selling agent |
| Searches | For seller's side |
| Council tax (outstanding) | If arrears |
| Factor / common charges | If arrears |
| Land Register fee | Registration of Disposition |
| Land and Buildings Transaction Tax | None for seller |
| Capital Gains Tax | Seller's personal tax liability (solicitor may require clearance) |

**Net due to Seller** = Total credits − Total debits

### Key Scottish-Specific Items

| Item | Notes |
|---|---|
| LBTT | Buyer pays; rates depend on property value (residential: up to 2% for £145K to £250K; 5% £250K to £325K; higher bands above; ADS, Additional Dwelling Supplement 6% since Dec 2024) |
| ADS (Additional Dwelling Supplement) | 6% on additional homes (since 5 Dec 2024) |
| Land Register fee | Fee for recording the Disposition in the Land Register (varies by value) |
| Council Tax | Scotland uses council tax bands A to H; pro-rated at settlement |
| Home Report | Seller must provide (includes single survey, energy report, property questionnaire) |
| Missives | Concluded missives = binding contract (not exchange of contracts) |
| Searching | Property search + Personal search (for bankruptcy / diligence) |
| Disposition | Deed transferring title (signed by seller before settlement) |
| Standard Security | Scottish mortgage document (registered with Land Register) |
| Completion | Also called "settlement" - funds and documents exchanged |
| Entry | Usually date of settlement (keys, possession, risk pass) |
| Escrow | Yes, possible (but rarely used, solicitor handles funds) |

### Proration Methods (Scotland)

| Item | Method |
|---|---|
| Council Tax | Daily rate (based on annual band); seller pays through day before entry |
| Factor / Service Charges | Annual charge; seller pays for period before settlement |
| Water charges | Included in council tax in Scotland |
| Ground burden / feu duty | Rare (abolished); only surviving ground burdens remain |

### Regulatory Context

- **Revenue Scotland** - administers LBTT
- **Registers of Scotland** - Land Register, General Register of Sasines, and other property registers
- **Law Society of Scotland** - Solicitors regulated; mandatory practice rules on conveyancing, financial handling, and CPD
- **Council of Mortgage Lenders (CML)** / **UK Finance** - Lender requirements (Standard Security, insurance, survey)
- **Home Report** (Scotland) - Legal requirement since 2008

### LBTT Return

- Buyer (or solicitor on their behalf) submits LBTT return to Revenue Scotland online, Must be done within **30 days** of completion date (not before)
- LBTT is payable on completion; solicitor typically handles via cleared funds, Penalties for late filing

### Key Differences Summary, US vs Scotland

| US (HUD/ALTA) | Scotland |
|---|---|
| HUD-1 / Closing Disclosure | Completion Statement |
| ALTA Settlement Statement | Solicitor-prepared statement |
| Title Insurance (settlement agent) | No title insurance, reliance on public registers |
| Escrow / Title company | Solicitor acts as settlement agent |
| Transfer taxes (county/state) | LBTT (national tax, collected by Revenue Scotland) |
| Recording fees (state/county) | Land Register registration fee |
| Prorated property taxes | Prorated council tax |
| HOA fees / estoppel | Factor fees / common charges (tenement management scheme) |
| 1099-S (seller) | No equivalent reporting requirement |
