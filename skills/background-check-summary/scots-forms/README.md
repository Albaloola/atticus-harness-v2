# Scottish/UK Forms, Background Check Summary

## Key Forms & References

| Purpose | Form / Reference | Issuer |
|---------|-----------------|--------|
| Basic DBS Check | DBS Basic application | Disclosure and Barring Service |
| Standard DBS Check | DBS Standard application | DBS |
| Enhanced DBS Check | DBS Enhanced application | DBS |
| PVG Scheme (Scotland) | PVG application (for regulated work with children/vulnerable adults) | Disclosure Scotland |
| Basic Disclosure (Scotland) | Basic Disclosure application | Disclosure Scotland |
| UK Sanctions List Search | https://search-uk-sanctions-list.service.gov.uk | FCDO / HM Treasury |
| Right to Work Check | Right to Work checks (Home Office) | UKVI |
| Equality Act 2010 | EHRC Code of Practice | Equality and Human Rights Commission |
| DBS Code of Practice | DBS Code of Practice | DBS |
| Data Protection Act 2018 | ICO guidance on criminal records checks | Information Commissioner's Office |
| Rehabilitation of Offenders Act 1974 | Exceptions Order 1975 | Legislation.gov.uk |
| Criminal Record Certificates | Police Act 1997 (Part V) | Legislation.gov.uk |

## Scotland-Specific

- **Disclosure Scotland**: https://www.mygov.scot/disclosure-types (for Basic, Standard, Enhanced, and PVG)
- **PVG Scheme**: https://www.mygov.scot/pvg-scheme (Protecting Vulnerable Groups)
- **Scottish Credit and Qualifications Framework (SCQF)**: For education verification
- **Companies House**: For directorship checks (https://find-and-update.company-information.service.gov.uk)
- **Insolvency Service**: For bankruptcy/director disqualification checks
