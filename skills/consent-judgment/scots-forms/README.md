# Scottish/UK Forms, Consent Order / Tomlin Order

## Key Forms & References

| Purpose | Form / Procedure | Jurisdiction |
|---------|-----------------|--------------|
| Tomlin Order (stay proceedings) | Standard Tomlin Order + Schedule | England & Wales |
| Simple Consent Order | CPR Part 23 application | England & Wales |
| Consent Decree (Scotland) | Consent decree recorded in Court of Session or Sheriff Court | Scotland |
| Interdict (Scotland) | Petition for interdict / crave in summons | Scotland |
| Injunction (England/Wales) | Claim Form / Application Notice | England & Wales |
| Administrative Consent | Court of Session practice note | Scotland |

## Scotland-Specific

- **Court of Session**: Consent decrees recorded in Outer House; crave for interdict in summons
- **Sheriff Court**: Consent decrees in Ordinary Cause or Simple Procedure
- **IPEC** (Intellectual Property Enterprise Court): Has own consent order procedure
- **Patents Court (Chancery Division)**: Standard consent order for patent settlements
- **Scottish Interdict Procedure**: Breach of interdict is a separate civil contempt process (not criminal)
- **Without prejudice privilege**: Common law privilege protecting settlement negotiations
- **Calderbank offers**: Costs protection in settlement negotiations

## Key Statutory References

- Civil Procedure Rules (CPR) Part 40 (Judgments and Orders)
- CPR Practice Direction 40B, Court of Session Rules Chapter 41 (Decrees)
- Sheriff Court Ordinary Cause Rules Chapter 13-15
- Trade Marks Act 1994 (licensing and assignment provisions)
- Patents Act 1977 (licensing provisions)
