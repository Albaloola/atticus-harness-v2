## Key Legislation

- Civil Litigation (Expenses and Group Proceedings) (Scotland) Act 2018 (asp 10)
  - Part 4 (ss.19-24) - Group Proceedings
  - s.20 - Definition of opt-in and opt-out proceedings
  - s.21 - Court of Session rule-making powers
  - s.22 - Permission requirement
  - s.23 - Remedies available
  - s.24 - Expenses (costs)
- Rules of the Court of Session 1994
  - Ch. 26A, Group Proceedings (inserted by SSI 2020/208)
- Prescription and Limitation (Scotland) Act 1973
- Scottish Civil Justice Council (procedural rules development)

## Key Principles

| Concept | Scottish Position |
|---------|-------------------|
| Opt-in / Opt-out | Opt-in only (current rules) |
| Court | Court of Session only |
| Permission required | Yes (court permission required) |
| Group definition | Defined in permission order |
| Prescription tolling | Not automatic for non-opt-ins |
| Damages | Compensatory only (no punitive) |
| Expenses | Judicial expenses follow the event |
| Third-party funding | Permitted but regulated |
