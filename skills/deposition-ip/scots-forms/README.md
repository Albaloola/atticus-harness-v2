# Scottish Forms, IP Litigation (Deposition Equivalent)

## UK / Scotland Key Forms & Resources

| Purpose | Form / Resource |
|---------|----------------|
| Commission and diligence | Court of Session, specification of documents / commission |
| IP proceedings (Court of Session) | Outer House, Commercial Court (patent/trade mark) |
| Patents Court (Scotland) | Court of Session, exclusive jurisdiction for patents |
| Proof (civil trial) | Proof in Court of Session or Sheriff Court |
| Reclaiming motion (appeal) | Inner House, Court of Session |
| UKIPO patent proceedings | UKIPO hearings, patents, designs, trade marks |
| Specification of documents | Order for recovery under Court of Session Rules Ch 35 |
| Interdict (interim/perpetual) | Court of Session / Sheriff Court |
| Expert witness guidance | Rules of the Court of Session Ch 37 (A) |
| Pre-action protocol (IP) | Pre-action protocol for intellectual property disputes |
