OSCOLA citation resources. The OSCOLA 5th edition guide is available free at https://www.law.ox.ac.uk/oscola. For Scottish case citation specifically, see https://library.rgu.ac.uk/oscola/cases-scotland. Session Cases (SC) is the primary Scottish law report series. Scottish neutral citations: CSOH (Outer House), CSIH (Inner House), SAC (Sheriff Appeal Court), UKSC (UK Supreme Court).
