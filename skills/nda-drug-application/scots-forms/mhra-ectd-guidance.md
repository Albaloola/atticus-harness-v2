# MHRA eCTD Guidance for Marketing Authorisation Applications

## Key Guidance

- **National Assessment Procedure**: https://www.gov.uk/guidance/national-assessment-procedure-for-medicines
- **eCTD Guidance for IRP**: https://www.gov.uk/government/publications/international-recognition-procedure/ectd-guidance-for-irp-mas-and-lifecycle
- **Apply for MA**: https://www.gov.uk/guidance/apply-for-a-licence-to-market-a-medicine-in-the-uk
- **Fees**: MHRA fees webpage

## Routes

1. **National Assessment** - for new active substances, biologics, orphans
2. **International Recognition Procedure (IRP)** - relying on Reference Regulator (FDA, EMA, Swissmedic, etc.)
3. **Access Consortium** - work-sharing with Australia, Canada, Singapore, Switzerland

## eCTD Format

- EU format eCTD with UK-specific Module 1
- Technical validation via Lorenz Docubridge, MHRA Submissions Platform only

## Key Legislation

- Human Medicines Regulations 2012 (SI 2012/1916)
- Paediatric Investigation Plans (PIPs)
- Orphan medicinal products designation
