| Purpose | Form / Resource |
|---------|----------------|

| POCA 2002 (confiscation) | Proceeds of Crime Act 2002, Part 7 |
| Terrorist Financing | Terrorism Act 2000 (as amended) |
| OFSI sanctions | Office of Financial Sanctions Implementation, consolidated list |
| FCA registration | FCA Gateway / Connect system |
| HMRC MLR registration | HMRC Money Laundering Registration portal |
| Beneficial ownership | PSC Register (People with Significant Control) via Companies House |
| CDD / EDD forms | Firm-specific; no prescribed HMRC/FCA forms |
| SAR reporting | SAR Online via National Crime Agency (NCA) |
