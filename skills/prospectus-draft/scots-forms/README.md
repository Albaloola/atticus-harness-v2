# Scots Forms Reference, UK Prospectus Draft

## Regulatory Forms

| Form | Description | Source |
|------|-------------|--------|
| FCA Registration Document | Prospectus registration (current regime) | FCA Handbook |
| AIM Admission Document | AIM market admission document | AIM Rules |
| FCA Technical Note | Prospectus contents checklist | FCA website |
| UKLR Disclosure Checklist | UK Listing Rules disclosure | FCA Handbook |
| Director's Declaration | Responsibility statement | FCA Handbook |

## Key Legislation

- Financial Services and Markets Act 2000 (FSMA)
  - s.85 - Prospectus requirement
  - s.90 - Compensation for misleading statements
  - s.90A, Liability for published information, Public Offers and Admissions to Trading Regulations (POATRs) (from Jan 2026)
- UK Market Abuse Regulation (UK MAR)
- UK Prospectus Regulation (current/transitional rules)
- Companies Act 2006
- FCA Handbook (including UKLR, DTR, PRR)
