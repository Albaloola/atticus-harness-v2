# Scottish Forms, Charity / Nonprofit Compliance

## UK / Scotland Key Forms & Resources

| Purpose | Form / Resource |
|---------|----------------|
| OSCR Annual Return | Online via OSCR portal |
| Charity registration | OSCR, charity application form |
| SCIO constitution | OSCR model constitution templates |
| Scottish charitable company | Companies House annual accounts + OSCR annual return |
| OSCR Monitoring Return | OSCR online monitoring questions |
| Gift Aid | HMRC Gift Aid claims, charitable donations |
| HMRC charity registration | HMRC charity reference and Gift Aid setup |
| OSCR Trustee Declaration | Annual self-declaration for charity trustees |
| Fundraising (Scotland) | Fundraising Regulator (Scotland) - code of practice |
| Public Collections | Section 6 Certificate, local authority street/trade collections |
| OSCR accounts guidance | OSCR: Charities SORP (FRS 102) |
| Scottish Charity Register | OSCR: https://www.oscr.org.uk |
